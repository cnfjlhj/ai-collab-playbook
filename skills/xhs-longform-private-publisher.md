# 小红书长文私密发布

对应本地 skill：`xhs-longform-private-publisher`

适用场景：
- 已经有 Markdown 长文
- 已经有配图
- 想尽量无损搬运到小红书
- 需要以私密方式发布，而不是直接公开

输入：
- 原始 Markdown 正文
- 配图路径
- 小红书创作者平台登录态

输出：
- 已排版并上传的长文内容
- 插图插入结果
- 私密发布后的验收记录

边界：
- 它偏“长文 + 私密 + 尽量不改写”的忠实搬运场景
- 真实发布依赖平台页面、登录状态与人工验收

独立仓库：<https://github.com/cnfjlhj/xhs-longform-private-publisher>
